Descripción: Texto.hs - Módulo para darle formato al texto y poder
comparar el nombre de la película con la respuesta del usuario sin que
importen espacios, mayúsculas y puntuación.

Reading.hs - Módulo que se encarga de leer el archivo con los nombres
de las películas y convertir los datos en una lista de haskell

Menu.hs - Módulo que se encarga de la interacción con el usuario.

quitar los artículos, ese es el propósito del método remplaza en
Texto.hs, pero no se me ocurrío cómo hacer para únicemente elimine
artículos y no letras separadas por espacios. Así que no está
implementado en el código, pero decido dejarlo en el código.
